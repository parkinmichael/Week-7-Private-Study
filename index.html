Lists

A list is a data structure containing elements all of the same data type and you may find duplicates in them. They are an example of a dynamic data structure meaning that their length can change throughout the runtime of a program and it is possible for the structure to'overflow' if it ecceeds the amount of memory that it's allocated.

One way to implement a list into java is to use the ArrayList function which can be used by importing the ArrayList library. A number of different operations can be performed on an ArrayList. These are shown in the table below.

The examples are based on the following list with the identifier myList: 45, 66, 41, 13, 19, 8

List Operation | Description | Example | Output

isEmpty() | Test for empty list | myList.isEmpty() | False add() | Appends the specified item to the end of a list | myList.add(5) | 5 remove() | Removes the element at the specified position of the list. Shifts any subsequent elements to the left | myList.remove(5) | n/a size() | Returns the number of elements in the list | myList.size() | The length of the list get() | Gets the element of a specified position in the list | myList.get(5) | Whatever is at the specified index clear() | Removes all the items from the list | myList.clear() | n/a no more elements contains() | Returns true if this list contains the specified element | myList.contains(5) | True if the list contains 5
